Indicator relevance vs oracle BUY/SELL signal

dir_accuracy%: of the bars the indicator actually moved on a BUY/SELL candle, % it moved the same way the signal called (50% = coin flip). corr_value / corr_change: Pearson correlation of raw value / bar-to-bar change vs the oracle's net edge.

Condition combination search

precision%: of the candles where ALL conditions in the combo were true, what % actually got that oracle signal. lift: precision / base rate (1.0x = no better than guessing).

Price action signal evaluation

hit_rate: % of fires that were profitable N candles later. avg_R: average forward move in ATR units. expectancy_R: the number that matters - positive means a real edge, negative means it loses on average.
